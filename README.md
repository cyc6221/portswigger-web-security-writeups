# PortSwigger Web Security Writeups

## 簡介

這個 repository 用來整理 PortSwigger Web Security Academy 的 lab writeups。內容依照弱點主題分類，每個主題目錄底下直接放對應的 Markdown writeup，方便閱讀、搜尋和後續擴充。

這些筆記主要用於學習、複習和紀錄解題流程。所有 lab 連結與題目內容皆屬於 PortSwigger Web Security Academy。
